# Proxy Switcher

System-wide IP switcher for Linux. Routes all system traffic through user-selected public proxies via a TUN device.

## Mission

Provide enterprise-grade IP rotation for any Linux application without per-app configuration — works at the network layer like a VPN.

## Key Features

- TUN-based traffic interception (system-wide, no app config)
- Public proxy scraping from multiple sources
- HTTP/HTTPS/SOCKS4/SOCKS5 support
- Health checking & ranking
- Manual proxy switch + auto-rotation
- CLI + Web UI control
- DNS leak prevention
